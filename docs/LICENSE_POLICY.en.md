# Licensing Policy (GPL)

Chinese primary version: [LICENSE_POLICY.md](LICENSE_POLICY.md)

## Selection

This project is licensed under `GPL-3.0-or-later`.

## Practical Effects (short)

- Commercial use is allowed (including paid distribution)
- Modification and redistribution are allowed
- If you distribute derivative works, source code must be provided under GPL terms

## About “restrictions”

The key GPL restriction is copyleft on distributed derivatives.

If you want:

- Open-source community edition
- Closed-source paid commercial edition

Typical approach:

- Keep clear copyright ownership
- Use a Contributor License Agreement (CLA)
- Apply dual licensing (GPL + commercial license)

## Note

This file is an engineering-oriented summary, not legal advice.
