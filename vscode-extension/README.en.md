# AIStack VSCode Extension

Control panel for AIStack orchestration.

## Features

- Brain/Worker provider config
- API URL/model/API key setup
- Full automation mode
- Dangerous permissions toggles (codex/claude/gemini)
- Run roadmap init / task package generation / worker execution

## Build

```bash
cd <repo-root>
npm install
npm run build

cd vscode-extension
npm install
npm run compile
```

Press `F5` in VSCode from `vscode-extension/` to start Extension Development Host.
