# AIStack model-router MCP

TypeScript/Node MCP server for multi-provider orchestration.

## Build

```bash
cd <repo-root>
npm install
npm run build
```

## Run

```bash
node dist/mcp/model_router/server.js
```

## Tools

- `model.one_shot`
- `model.brainstorm`
  - participants
  - debate_rounds
  - synthesis_by
