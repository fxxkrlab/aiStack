# AIStack model-router MCP

中文主文档。English version: [README.en.md](README.en.md)

`model-router` 是统一的多模型 MCP 入口。

## 构建

```bash
cd <repo-root>
npm install
npm run build
```

## 启动

```bash
node dist/mcp/model_router/server.js
```

## 可用工具

- `model.one_shot`
- `model.brainstorm`
  - participants
  - debate_rounds
  - synthesis_by
